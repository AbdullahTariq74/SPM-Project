# Tech Stack

## Languages
- JavaScript (Node.js)

## Runtime & Frameworks
- Backend: Node.js, Express
- Frontend: Vite, React
- Styling: TailwindCSS

## Dependencies & Configuration
- Containerization: Docker, docker-compose
- Database: PostgreSQL (15-alpine)

## Environment Variables
- `DB_HOST` (mapped in Docker to `db`)
- `POSTGRES_DB` (SPM_db)
- `POSTGRES_USER`
- `POSTGRES_PASSWORD`
